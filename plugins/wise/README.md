# Wise Codex Plugin

This plugin packages the `wise-api-cli` skill for Codex and points agents to the `@brahimhamichan/wise-cli` command surface for Wise Platform API work.

The plugin supports workflows for:

- Wise token setup and auth checks
- Profiles and balances
- Recipients and quote requirements
- Quote previews and creation
- Transfers, receipts, and statements
- Raw Wise API requests through `wise api`
